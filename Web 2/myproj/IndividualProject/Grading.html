Grading

  • CSS/Styling: I used boot strap for this project however I modified the variables and recreated my modified version to produce the site theme. You will see intermittent inline styles and the reason for that is because those are needed to override the boot strap theme in specific area's of the website. Check out the modeling I used in the gallery for the pictures. When you click on containers the picture should pop out and convert into a slideshow effect with a next and previous link.
  • Javascript/Jquery: I used jquery in the gallery section in order to create the popup and slideshow effect. Boot strap provides some built in class to help create the gallery and pop up but jquery was needed to make the previous and next buttons work correctly. I also used the provided javascript components by boot strap.
  • Other Notes: For speed the boot strap components and jquery are loaded from a CDN provider and they are the last thing to load because you want to be able to see the page as quickly as possible. The boot strap components load asyncroniously but jquery does not because since boot strap depends on jquery it must be fully loaded before we start loading the boot strap components. I also used a call to the google map api on the contact page to load a map of the coordinates that I specify. This loads a map of wherever I choose onto the page without having to save it as an asset so we can save space and let google do the work instead of us. You will also see in comments a script that it is loaded only if the user is viewing this webpage in IE 9. This is needed to make hmtl5 work on IE 9.
